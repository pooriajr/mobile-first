mobile-first.css is a tiny stylesheet that makes your <body> element resemble a mobile phone.

Why?

When building on desktop it can be easy to accidentally slip into a desktop-first mentality. This is merely a visual cue to use during development to keep you in the mobile-first mindset.

Usage

Paste this tag in the <head> element of the page:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/pooriar/mobile-first/mobile-first.min.css">

That's it! When you don't need it anymore, delete the tag.

How does it work?

It adds a few styles to the HTML and <body> tag, nothing else.

The result is a viewport with a fixed width and height, like a phone. It only scrolls vertically, and has a 16:9 aspect ratio, which is the most common in phones.

Variants

The simple default is the most practical, but there are also 2 variants for fun. Activate them by adding their class to the body element.
