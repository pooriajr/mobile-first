mobile-first.css

mobile-first.css is a tiny css file that styles the <HTML> element of your page to resemble a mobile phone.

Why?

When building on desktop it can be easy to accidentally slip into a "desktop-first" mentality. This is a visual cue to keep you in the mobile-first mindset right from the start of the project.

Usage

Paste this tag in the <head> element of the page:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/pooriar/mobile-first/mobile-first.min.css">

When you don't need it anymore, delete the tag. That's it!
