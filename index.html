mobile-first.css is a tiny stylesheet that makes your desktop view resemble a mobile view.

Why?

When building on desktop it's easy for me to accidentally slip into a desktop-first mentality. This is a visual cue to use during development to stay in the mobile-first mindset.

Usage

Paste this tag in the <head> element of the page:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/pooriar/mobile-first/mobile-first.min.css">

That's it!

What does it do exactly?

It adds a dozen styles to the <HTML> and <body> tags.

The result is a view with a fixed width and height, like a phone.

It only scrolls vertically, and has a 16:9 aspect ratio, which is the most common in phones.
